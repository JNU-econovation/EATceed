## Project Duration

Ver 1.0 : 2023.10.01 ~

## Feature Overview

- 사용자가 섭취한 음식을 기반으로 칼로리와 영양 성분을 계산
- 칼로리 관리와 체중 증가 목표를 달성하기 위한 진행 상황을 추적
- 알림을 통해 사용자가 음식을 설정한 시간에 섭취하도록 유도하여 건강한 체중 증가 지원
- 칼로리 캘린더를 통한 일일 권장 섭취 칼로리 달성률을 확인
- 챗봇을 이용한 음식 추천
